## Создать массив

```c++
int a[3];           // массив int-ов длины 3
std::string ss[3];  // массив string-ов длины 3
```

## INDEX:

```c++
int a[3];  
a[4] = 0;  // ошибки не будет, только если не включены санитайзеры (будет портиться память)
```
